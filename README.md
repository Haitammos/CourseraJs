# Galerie de Photos Interactive

Une galerie de photos accessible avec navigation au clavier et à la souris.

## Fonctionnalités

- Navigation accessible (souris et clavier)
- Six images avec texte alternatif
- Design responsive
- Validation W3C, WAVE et aXe

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
